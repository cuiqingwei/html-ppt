# HTML PowerPoint Presentation

## Project Documentation

This project allows you to create PowerPoint presentations using HTML and CSS. The aim is to provide a lightweight and easy-to-use tool for making presentations that are web-based and accessible across different devices.

### Features
- Create slides using HTML
- Style slides with CSS
- Export options for different formats
- Responsive design for viewing on multiple devices

## Usage Guide

1. **Installation**:
   To get started, clone the repository:
   ```bash
   git clone https://github.com/cuiqingwei/html-ppt.git
   cd html-ppt
   ```

2. **Creating a Slide**:
   Each slide can be created as a separate HTML file. Configure your slides using standard HTML elements and CSS for styling.

3. **Running the Presentation**:
   Open the main HTML file in your web browser to start the presentation.

4. **Exporting to PowerPoint**:
   Use the provided scripts to export your presentation to a PowerPoint format, if available.
